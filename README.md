# 고2 진학 상담 분석 프로그램 (로컬 Streamlit)

로컬 환경에서 실행하는 Streamlit 기반 진학 상담 분석 프로그램입니다.

## 핵심 특징
- 졸업생 엑셀 업로드
- 현재 고2 상담자료 PDF/이미지 업로드
- 자동 추출 후 확인/수정
- 유사 졸업생 매칭
- 전형 적합도 분석
- 학생용 보고서 HTML/PDF 출력
- 현재 학생 원데이터 저장 안 함

## 실행 방법
```bash
pip install -r requirements.txt
streamlit run app.py
```

## 권장 운영 방식
- 개인 PC 또는 학교 내부 PC에서만 로컬 실행
- 현재 학생 자료는 세션 종료 시 폐기
- 결과 PDF만 필요할 때 저장


## 로그인
프로그램 첫 화면에서 로그인을 거쳐야 사용할 수 있습니다.

기본 로그인 정보 파일 위치:
- `sample_data/login_users.xlsx`

권장 열 이름:
- `아이디`
- `비밀번호`

예시 파일에는 테스트용 계정 1개가 포함되어 있습니다.
